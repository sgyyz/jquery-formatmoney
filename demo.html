jQuery Format Money Plugin Demo - format money as you like.

You can download or fork this plugin here. Or if you find any issues or you have any idea, please put it here.

Demo 1

jQuery Format Money can used to format the text with your configuration.

For Example

Format the text 21.1523, with prefix $ and 2 decimal. 

<span class="text-money">21.1523<span>

...
$('.text-money').formatmoney({'prefix': '$'});

Result: 21.1523

Demo 2

You can also use this plugin in your input text fields. When the page loaded, the plugin will help you format the default value. If this input field's value is not defined, it will be the zero with configuration format. Such as: 

<input type="text" class="demo-2-input-1"/>

...
$('.demo-2-input-1').formatmoney({
	'prefix': '$',
	'precision': 3
});

Result:

Input field with default value: $15231, only allow integer number in the input field. It should display like $15,231.

<input type="text" class="demo-2-input-2" value="$15231"/>

...
$('.demo-2-input-2').formatmoney({
	'prefix': '$',
	'precision': 0
});

Result:

Note: In the input field, it also allow the key events, so every input will add at the end of the value. As well, every delete will remove the last value.

Options for customer definition

Option Name Default Value Description
prefix non Define the money prefix like $, ï¿¥ and other symbol. After set this option, the value of you money will add it at the first position.
thousands , The thousands symbol, it will help user to identify the value quickly and beauty.
decimal . User can set this symbol to separate the integer part and decimal part.
precision 2 This option used to define the decimal part length. The default setting will format the value like 1.22. If you set it to 0, it will only allow the integer numbers.
